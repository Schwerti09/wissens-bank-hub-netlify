Quickcheck Empfehlungen

1) Sofortmaßnahmen

Keys

API-Keys rotieren

Alte Keys deaktivieren. Neue Keys nur als Secrets.

Netz

Firewall

Nur Ports offen, die du brauchst. Default: alles zu.

Zugriff

SSH härten

Key-Auth, Root-Login aus, Fail2ban, 2FA.

2) Standard-Setup

  1. Ubuntu LTS installieren, sofort Updates.
  2. Neuen Admin-User anlegen, Root-SSH deaktivieren.
  3. UFW aktivieren: 22 (optional), 80/443, sonst nix.
  4. Reverse Proxy + HTTPS + HSTS.
  5. Secrets: nur ENV/Secret Manager.
  6. Backups: täglich + Restore-Test.
  7. Monitoring: Uptime + Logs + Alerts.
